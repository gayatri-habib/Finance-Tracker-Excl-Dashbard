# Finance-Tracker-Excl-Dashbard
Personal Finance Tracker Dashboard in Excel to analyze income, expenses, and savings trends.

## Project Overview
This project is a personal finance tracking dashboard built using Microsoft Excel to monitor income, expenses, and savings.

## Tools Used
- Microsoft Excel
- Pivot Tables
- Pivot Charts
- Excel Dashboard Design

## Dashboard Features
- Monthly income tracking
- Expense breakdown by category
- Savings analysis
- Living vs discretionary expense comparison
- Interactive filters by month and year

## Dataset Sheets
- Bank Transactions
- Categories
- Monthly Data
